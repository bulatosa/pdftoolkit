# PDF Toolkit Pro

A modern Flutter PDF utility app template built for marketplaces and production-ready customization.

## Features

* Image to PDF
* PDF to Images (PNG Export)
* Merge PDF Files
* Split PDF Files
* Extract PDF Pages
* My Files Manager
* Open & Share Exports
* Responsive Phone & Tablet Layouts
* Material 3 UI
* Offline Workflow

## Getting Started

```bash
flutter pub get
flutter run
```

## Build Release APK

```bash
flutter build apk --release
```

## Tech Stack

* Flutter
* Dart
* Material 3
* Syncfusion PDF
* Shared Preferences

## Notes

* No backend required
* No API keys required
* Works completely offline
* All file operations are performed locally on the device
